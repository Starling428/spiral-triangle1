Main.exe draws a picture of rotating triangle.

Usage of main.exe:
  -degree float
        Rotate angle (degrees) (default 63)
  -inversion
        Invert colors
  -iter int
        Quantity of iterations (default 55)
  -length int
        Length of the base line (default 30)
  -name string
        Invert colors (default "out.png")
  -scale float
        Scale multiplier (default 1.06)
  -size int
        Length of the image side (the image is a square (default 300)
